# Inner Play Landing Page

Public landing page for Inner Play by Moving Truth at `inner-play.movingtruth.com`.

The site is static, dependency-free, accessible, responsive, and grounded in Inner
Play's creative charter: believable tactile response, sensory choice, and play without
pressure.

`assets/icon-placeholder.png` is a provisional website concept, not the approved or
shipped application icon. Replace it everywhere only after the final Inner Play icon
has been selected.
